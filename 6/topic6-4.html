💳 Online Payment Systems

📌 Overview

Online payment systems allow users to pay for goods and services over the internet by sending payment data securely to processing systems.

1

🌐 Third-Party Payment Processors

Use accounts to send and receive money
  • Examples: PayPal, Skrill
  • Use email for identification
  • Fast and convenient for online shopping
💡 Exam Tip: Third-party = middle service for payments
2

💳 Bank Cards

Direct payment method

Used to pay online or in shops using card details.

  • Card number
  • Expiry date
  • Name on card
  • Security code (CVV/CVC)
3

📡 Contactless (NFC)

Tap-to-pay technology
  • Uses near field communication
  • No PIN needed for small payments
  • Has a maximum limit for safety
💡 Exam Tip: NFC = short-range wireless payment

⚠️ Security Notes

  • Card number ≠ bank account number
  • Security code protects transactions
  • Be careful of fraud and data theft

🛡️ Payment Security Systems

Protect online transactions
i

HTTPS

Encrypted communication

Encrypts data between browser and server to keep payment details secure.

ii

Authentication Systems

Verify user identity
  • Password or PIN
  • One-time password (OTP)
  • 3D Secure (e.g. SecureCode)
iii

Encryption

Scrambles data

Protects payment data so it cannot be read if intercepted.

iv

Tokenisation

Replaces card details

Replaces real card data with a secure token to reduce risk of theft.

v

Fraud Detection

Monitors suspicious activity

Systems detect unusual transactions and block possible fraud.
