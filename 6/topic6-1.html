⚠️ Risks to Data and Information

⚠️ 4 Main Risks to Data

🚫 Unauthorised Access

🦠 Malware

❌ Accidental Deletion

🕵️ Theft of Personal Data

I

🚫 Unauthorised Access

Access by users without permission
  • Hackers try to log in using many password attempts
  • Weak security makes systems vulnerable
  • Devices can be used as botnets
💡 Exam Tip: Unauthorised = no permission
II

🦠 Malware

Malicious software that causes harm
  • Deletes files or damages systems
  • Can display messages or control devices

Types of Malware

1

🦠 Virus

Attaches to files/programs

Spreads when infected files are opened or shared.

2

🐴 Trojan

Disguised as legitimate software

Tricks users into installing it, then causes harm.

3

🪱 Worm

Self-replicating malware

Spreads automatically across networks without user action.

4

📢 Adware

Displays unwanted adverts

Shows pop-ups or advertisements, often slowing down the system.

5

👁️ Spyware

Secretly monitors users

Collects personal data such as passwords and browsing activity.

6

🔐 Ransomware

Blocks access to data

Demands payment to unlock files or systems.

💡 Exam Tip: Virus = attaches, Worm = spreads, Trojan = tricks
III

❌ Accidental Deletion

User mistakes
  • Pressing wrong keys
  • Formatting wrong storage device
  • Power failure during use
