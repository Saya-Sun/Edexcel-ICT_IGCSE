III

🕵️ Theft of Personal Data

Stealing sensitive user information

Criminals use different methods to steal personal data such as passwords, bank details, and personal information.

1

🎣 Phishing

Fake messages to trick users
  • Emails or messages pretending to be real organisations
  • Ask for login or payment details
  • Often include links to fake websites

Spear phishing: targeted at specific users

Smishing: phishing via SMS

2

🌐 Pharming

Redirects users to fake websites

Pharming is a technique used to redirect users to fake websites that look like real ones in order to steal personal data.

⚠️ Common Trick

Fake URLs look very similar to real ones (e.g. bank.com → bankk.com).

🔹 Type 1: DNS Redirection
  • Internet traffic is redirected to a fake website
  • Caused by changing DNS settings or using malware
  • User thinks they are on the real website
🔹 Type 2: Fake URL (Typo Attack)
  • Fake website URL looks similar to real one
  • User makes a typing mistake and visits fake site
  • Example: moneybank.lk → moneybamk.lk
💡 Exam Tip: Pharming = redirect or fake URL trick
💡 Exam Tip:
Phishing = trick message,
Pharming = automatic redirection to fake website
