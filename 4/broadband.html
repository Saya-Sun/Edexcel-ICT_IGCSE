🌐 Broadband & Cellular Networks

🌐 Broadband
Broadband is a high-speed internet connection provided through an Internet Service Provider (ISP).
  • Uses fibre optic cables (light signals)
  • Or copper cables (electrical signals)
  • Always-on connection (no need to reconnect)
  • High bandwidth → fast downloads and streaming
👉 Fibre optic is faster and more reliable than copper
📱 Mobile Broadband (3G / 4G)
Wireless internet connection using mobile phone networks.
  • No cables required → portable
  • Uses 3G, 4G (and newer 5G)
  • Useful when Wi-Fi is unavailable
  • Speed depends on signal strength and location
👉 Mobile network acts as the ISP
📡 Cellular Networks
A cellular network is another name for a mobile phone network.
  • Area divided into cells
  • Each cell has a base station (tower)
  • Devices connect to nearest tower
  • Automatically switches cells when moving
👉 Enables continuous connection while travelling
⚖️ Advantages & Limitations
  • Broadband: fast, stable, high bandwidth
  • Mobile: portable, flexible
  • Limitation: mobile depends on coverage
  • Limitation: slower than wired broadband
📘 Key Definitions
  • ISP: company providing internet access
  • Fibre optic: transmits data using light
  • Copper cable: transmits data using electricity
💡 Health & Safety
  • WHO: no proven health risks from wireless signals
  • Long-term effects still under research
  • Children may be more vulnerable
Wi-Fi wavelength is much longer than visible light → generally considered safe
