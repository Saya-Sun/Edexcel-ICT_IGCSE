📡 Speed & Volume of Data Transfer

📦 Overview
  • More data → longer transfer time if speed is low
  • Need faster transfer for modern applications
🌐 Bandwidth
Bandwidth is the amount of data transferred per second.
  • Higher bandwidth → more data transfer
  • Like a wider pipe carrying more water
🚀 Impact of Bandwidth
  • Faster uploads and downloads
  • Supports gaming and streaming
  • Handles large data quickly
🎬 Buffer
Temporary storage used before playback.
  • Video starts after buffer fills
  • Must refill continuously
  • If empty → buffering pause
👉 Data must download faster than playback
⏱️ Latency
Delay in sending data between devices.
  • Measured using ping
  • Lower latency → faster response
🎮 Impact of Latency
  • High latency → lag
  • Slow response in games
  • Delay in live streaming
⚠️ Factors Affecting Transfer
  • Interference / noise
  • Weak signal strength
  • Network congestion
  • Errors → data resent
