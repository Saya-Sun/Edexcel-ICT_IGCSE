📡 Speed & Volume of Data Transfer

📦 Overview
  • More data → longer transfer time if speed is low
  • Need faster transfer for modern applications
🌐 Bandwidth
Bandwidth is the amount of data transferred per second.
  • Higher bandwidth → more data transfer
  • Like a wider pipe carrying more water
🚀 Impact of Bandwidth
  • Faster uploads and downloads
  • Supports gaming and streaming
  • Handles large data quickly
🎬 Buffer
Temporary storage used before playback.
  • Video starts after buffer fills
  • Must refill continuously
  • If empty → buffering pause
👉 Data must download faster than playback
⏱️ Latency
Delay in sending data between devices.
  • Measured using ping
  • Lower latency → faster response
🎮 Impact of Latency
  • High latency → lag
  • Slow response in games
  • Delay in live streaming
⚠️ Factors Affecting Speed & Volume of Data Transfer
📦 Overview
Data transfer can be affected by factors that prevent bits from reaching their destination.
  • Errors may occur during transmission
  • Data must be resent
  • This slows down overall transfer speed
1️⃣ Transfer Method
  • Wireless uses limited frequencies
  • Copper cables carry more frequencies
  • More frequencies → higher bandwidth
👉 Wired connections usually provide higher bandwidth
⚠️ Some older wired methods may be slower than modern wireless
2️⃣ Interference
  • Electromagnetic signals disrupt data
  • Sources: routers, phones, microwaves, appliances
  • Causes errors → data resent → slower speed
Shielded cables use metal layers to reduce interference
3️⃣ Blockages
  • Walls and furniture weaken wireless signals
  • Reduces signal strength
  • Leads to lower bandwidth
4️⃣ Distance
  • Signal strength decreases over distance
  • Affects both wired and wireless
  • Weaker signal → slower data transfer
🧠 Quick Summary
  • Transfer method → affects bandwidth
  • Interference → causes errors
  • Blockages → weaken signal
  • Distance → reduces signal strength
