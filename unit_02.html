🧠 What is Software?

The programs that run on a computer are called software. Programs are sets of instructions that are carried out (executed) by the computer’s processor (CPU).

The processor controls all operations of the computer by following these instructions.

  • Some computers can run only one program at a time
  • Others can run multiple programs simultaneously (multitasking)

Software makes hardware useful. Without software, a computer cannot perform any task.

🖥️ Types of Computers (Based on Software)

1. Single-Purpose Computers

These computers are designed to perform only one specific task and run only one type of software.

  • Microwave ovens
  • Washing machines
  • ATM machines
2. General-Purpose Computers

These computers can perform many different tasks and run multiple types of software.

  • Desktop computers
  • Laptops
  • Tablets
  • Smartphones

📂 Types of Software

There are two basic groups of software:

⚙️ System Software
📱 Application Software

I. ⚙️ SYSTEM SOFTWARE

Programs that are designed to maintain or operate the computer system are known as system software.

Functions include:

  • Managing hardware
  • Controlling memory
  • Running applications
  • Providing a user interface

Types of System Software:

  • Operating Systems (OS)
  • Device Drivers
  • Utility Software

II. 📱 APPLICATION SOFTWARE

Application software (apps) allow users to produce digital products or carry out specific tasks that are not related to the operating system.

Examples of tasks:

  • Word processing
  • Spreadsheets
  • Presentations
  • Graphics editing

Apps are usually downloaded from servers on the internet or installed from storage media such as DVDs.

App stores check applications to ensure they are safe and do not harm devices or data.

Exam Tip: Always write the type of software (e.g. word processing software), not brand names.

III. 🔐 SOFTWARE LICENSING

A software licence is a legal agreement that allows users to install and use software.

Types of licences:

  • Freeware – free to use
  • Shareware – trial version
  • Open source – free and modifiable
  • Proprietary – paid and restricted

Software licensing rules can be complex, but they control how software can be used and distributed.

IV. 🔄 SOFTWARE UPDATES

Updates to software are released by developers to improve and fix software.

Reasons for updates:

  • Fixing bugs and security issues
  • Improving compatibility
  • Increasing performance
  • Adding new features
  • Improving usability

Updates are usually downloaded from the internet and can be automatic or manual.

⚠️ Always back up your data before updating to avoid data loss or system issues.
