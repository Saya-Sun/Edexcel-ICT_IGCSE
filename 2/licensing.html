🔐 SOFTWARE LICENSING

Understanding Legal Use of Software (IGCSE ICT)

🧠 What is a Software Licence?

A software licence is a legal agreement that allows users to install and use software. :contentReference[oaicite:0]{index=0}

It defines what users are allowed to do, such as:

  • Copy the software
  • Modify the software
  • Share the software
⚠️ Using software without a licence is called software piracy and is illegal.
Free / Open Source Proprietary

1. 🟢 Free & Open-Source Software

Users have the freedom to:

  • Study the program
  • Modify it
  • Copy it
  • Distribute it

Open-source software provides access to the source code.

KEY POINT: "Free" refers to freedom, NOT price.
Exam Tip:
Free = freedom
Open-source = source code available

Examples: Linux, Python, LibreOffice

These are flexible and customizable, making them popular for developers.

2. 🔵 Proprietary Software

Owned and controlled by a company or developer.

  • Users cannot modify the software
  • Source code is not available
  • Usage is restricted

Can be:

  • Paid software
  • Freeware (free but restricted)

Examples: Windows, Microsoft Office, Adobe Photoshop

These are popular due to professional support and ease of use.

📊 Comparison (Very Important for Exams)

Feature Free/Open Source Proprietary
Cost Free or paid Usually paid
Source Code Available Not available
Modification Allowed Not allowed
