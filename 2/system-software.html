⚙️ SYSTEM SOFTWARE

Detailed IGCSE ICT Notes | Full Explanation for Exam Success

🧠 Definition

System software consists of programs designed to maintain and operate a computer system. It acts as a bridge between hardware and application software, ensuring the computer functions correctly.

🛠️ A. Utility Software

Utility software is a type of system software that performs maintenance and optimisation tasks to keep a computer running efficiently.

1. 💾 Backup

Backup utilities create copies of files and programs. These backups can be automatic or manual.

  • Protects against data loss
  • Useful after system failure or virus attack
  • Types: Full, Incremental, Differential
Backups are usually scheduled when the system is not in use.
2. 🧩 Defragmentation

Data on a hard disk can become fragmented, meaning parts of a file are stored in different locations.

  • Slows down data access
  • Defragmentation reorganises data into proper order
  • Improves system speed
Mostly required for HDDs, not SSDs.
3. 🗜️ Compression

Compression reduces file size by removing repeated patterns of data. 

a8b6c2 (example of compressed data)
  • Saves storage space
  • Makes file transfer faster
  • Requires decompression before use
4. 💽 Formatting

Formatting prepares a storage device for use by setting up a file system.

  • Deletes all data on the disk
  • Allows user to choose file system
  • Used for new or reused drives
⚠️ Formatting permanently removes data. Be careful!

🖥️ B. Operating System (OS)

The operating system allows the user to control and manage the computer’s hardware and software.

Types of OS
  • Single-user operating systems
  • Network operating systems
Network OS can manage multiple users and shared resources.

⚙️ Functions of Operating System

1. 🧠 Memory Management

Allocates RAM to applications and frees it when not needed. Also manages virtual memory.

2. ⚙️ Resource Management

Allocates system resources such as CPU and printers. Uses queues if resources are busy.

3. 🔐 Security

Provides authentication using passwords, biometrics, or access cards. Includes firewalls.

4. 🖨️ Print Spooling

Stores print jobs in a queue and sends them to the printer one at a time.
