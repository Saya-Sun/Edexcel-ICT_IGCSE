📱 APPLICATION SOFTWARE

Complete Detailed IGCSE ICT Notes | Full Explanation

🧠 Definition

Application software (apps) allow users to produce digital products or carry out specific tasks that are not related to the operating system. :contentReference[oaicite:0]{index=0}

Apps can be downloaded from servers on the internet or installed from storage media such as DVDs. Many devices use app stores that check apps for quality and security before making them available.

Exam Tip: Always refer to the TYPE of application (e.g. word processing software), not brand names.

📂 A. Office Productivity Software

These applications are used to perform work-related tasks and are often available as a suite.

1. 📝 Word Processing
  • Create documents (letters, essays, reports, books)
  • Add images, tables, charts, hyperlinks
  • Spell check and grammar tools
  • Track changes and comments (collaboration)
2. 📰 Desktop Publishing (DTP)
  • Advanced page layout design
  • Used for magazines, brochures, posters
  • More control than word processors
3. 📊 Spreadsheet
  • Perform calculations using formulae and functions
  • Automatic updates when values change
  • Used for financial modelling and “what-if” analysis
4. 🗂️ Database (DBMS)
  • Store, search and manage data
  • Forms for input
  • Queries to filter data
  • Reports for output

🌐 5. Web Authoring

  • Create web pages using HTML
  • Web browsers display the pages
  • WYSIWYG editors allow design without coding

🎨 6. Image Editing

(a) Graphics Editors
  • Create vector graphics
  • Layers, shapes, colours
  • Resize and align objects
(b) Photo Editors
  • Edit photos
  • Adjust brightness and contrast
  • Apply filters and effects
  • Crop and resize

🎵 7. Sound Editing

  • Cut and join audio
  • Adjust volume and tempo
  • Add effects (echo, reverb)
  • Noise reduction and normalisation

📊 8. Presentation Software

  • Create slides with text, images, video
  • Add animations and transitions
  • Presenter notes and timings
  • Automatic slide transitions

🤖 Control Software

Control software uses sensor input, processes it, and sends commands to output devices.

  • Used in automation systems
  • Examples: lifts, robots, smart homes
  • Works with sensors and actuators
Example: Lift doors open automatically when sensors detect correct position.

📅 9. Project Management

  • Plan and track tasks
  • Identify dependencies between tasks
  • Shows critical path
  • Uses Gantt charts

💬 10. Communication Software

i. Web Browsers
  • View websites
  • Access internet services
ii. Email
  • Send/receive messages
  • Use attachments
  • Webmail or client software
iii. Social Media
  • Share content online
  • Access via apps or browsers
iv. SMS
  • 160-character messages
  • No internet required
v. MMS
  • Send multimedia (images, video)
vi. Instant Messaging
  • Real-time messaging
  • Typing indicators
  • Can send location data
