💾 Secondary Storage

IGCSE ICT - Complete Notes for Exam Success

📘 1. What is Secondary Storage?

Secondary storage (also called storage) is used to store data permanently. Unlike RAM, it does not lose data when power is turned off.

⚠️ Exam Tip: Do NOT confuse storage with memory (RAM/ROM).
  • Non-volatile (data is not lost)
  • Used for long-term storage

🎯 2. Purpose of Secondary Storage

📄 Store Files

Documents, images, videos saved for future use.

💻 Store Software

Applications are stored and loaded into RAM when opened.

⚙️ Virtual Memory

Acts as extra RAM when memory is full.

💻 3. Storage Devices

Storage devices can be internal or external.

  • Internal: Inside computer (connected to motherboard)
  • External: Connected via USB or wireless

🔹 Hard Disk Drive (HDD)

  • Uses magnetic storage
  • Contains spinning platters
  • Speed: 5400–7200 rpm
  • Seek time: 4–15 ms
✔ Cheap and large capacity
❌ Slower and can break (moving parts)

🔹 Solid State Drive (SSD)

  • Uses flash memory
  • No moving parts
  • Very fast and reliable
✔ Fast, durable, silent
❌ More expensive

🔹 Optical Disks

Type Capacity
CD 700 MB
DVD 4.7 GB – 18 GB
Blu-ray 25 GB – 50 GB

Backwards Compatibility: Blu-ray drives can read DVDs and CDs.

🔹 Magnetic Tape

Used for backups in large organisations. Very cheap but slow.

💽 4. Storage Media

  • Hard Disk: Magnetic platters
  • Optical: Laser technology
  • Flash: Electrical charges

⚙️ 5. Binary Storage

All data is stored in binary (0 and 1).

Media Type How Data is Stored
HDD Magnetic charge
Optical Surface bumps
Flash Electrical charge
Tape Magnetic polarity

📏 6. Storage Capacity

  • KB, MB, GB, TB
  • Blu-ray: 25–50 GB

🧠 7. Real World Example

The first disk drive (IBM 350) stored only 3.75 MB, while modern microSD cards can store 256 GB in a tiny size.

⚠️ 8. Common Exam Mistakes

  • Confusing RAM and storage
  • Forgetting SSD has no moving parts
  • Mixing optical and magnetic storage
