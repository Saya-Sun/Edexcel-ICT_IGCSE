📱 Multifunctional Devices & Convergence

IGCSE ICT - Complete Notes for Exam Success

📱 What is a Smartphone?

A smartphone is a mobile phone whose primary function is communication, but it can also perform many advanced tasks.

Functions:
  • 📸 Camera (photos & videos)
  • 🧭 GPS Navigation
  • 🎧 Music Player
  • 🎮 Gaming Device
  • 🏃 Fitness Tracker
A smartphone is a multifunctional device because it performs many tasks in one device.

🔹 Multifunctional Devices

A device that performs multiple functions.

Examples:
  • Smartphones
  • Tablets
  • Laptops
  • Smartwatches

🔄 What is Convergence?

Convergence is when multiple technologies are combined into one device.

Example: A smartphone combines a camera, GPS, music player, and phone.

💻 Device Convergence in Action

  • Laptops with touchscreens
  • 360° foldable laptops
  • Tablets with keyboards
These are called hybrid devices.

📊 Comparison

Device Features
Smartphone Portable, mobile network, smaller screen
Tablet Larger screen, sometimes no mobile network

🧠 Exam Key Points

  • Multifunctional device = one device, many functions
  • Convergence = combining technologies
  • Smartphones = best example
  • Reduces need for multiple devices
  • Differences between devices are decreasing
