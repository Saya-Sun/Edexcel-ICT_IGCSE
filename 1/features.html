IGCSE ICT

💻 Features of Digital Devices

Complete IGCSE Notes for Full Marks 🚀

📱 1. Portability

How easy a device is to carry.

  • Small size → more portable
  • Light weight → easier to carry
  • Battery-powered → highly portable
💡 Exam Tip: Always mention size + weight when defining portability.

⚡ 2. Performance

  • CPU speed → processing power
  • RAM → multitasking
  • SSD faster than HDD
  • Virtual memory used when RAM is full
💡 Exam Tip: RAM = temporary, Storage = permanent.

💾 3. Storage

  • Stores data permanently
  • SSD (fast) vs HDD (slower)
  • Cloud storage available

🖥️ 4. User Interface

  • CLI (text-based)
  • GUI (visual)
  • Menu-driven
  • Voice & Gesture interfaces

🌐 5. Connectivity

  • Wired = faster & stable
  • Wireless = convenient
  • Used for data sharing

💽 6. Media Support

  • Supports SD, microSD, DVD
  • Adapters improve compatibility

🔋 7. Energy Consumption

  • Low energy = longer battery life
  • Eco-friendly devices preferred

🔌 8. Expansion Capability

  • External drives & USB devices
  • PC = high expandability
  • Phones = limited expansion

🔐 9. Security Features

  • Passwords & PINs
  • Biometric authentication
  • Auto-lock systems
  • Physical locks
💡 Exam Tip: Always give at least 2 examples of security methods.
