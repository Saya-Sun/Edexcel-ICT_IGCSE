Secondary Storage

Complete Guide for Exam Success

1. What is Secondary Storage?

Secondary storage (storage) is used to store data permanently.

  • Non-volatile (data is not lost when power is off)
  • Used for long-term storage
📌 Exam Tip: Do NOT confuse storage with memory (RAM).

2. Purpose of Secondary Storage

  • Store documents for future use
  • Store applications (loaded into RAM when opened)
  • Used as virtual memory

3. Storage Devices

Storage devices store data and software.

  • Internal: Inside computer (connected to motherboard)
  • External: Connected via USB or wireless

4. Types of Storage Devices

4.1 Hard Disk Drive (HDD)
  • Uses magnetic storage
  • Spinning platters
  • 5400–7200 rpm
  • Seek time: 4–15 ms

Advantages:

  • Large capacity
  • Cheap

Disadvantages:

  • Slow
  • Can break (moving parts)
4.2 Solid State Drive (SSD)
  • Uses flash memory
  • No moving parts
  • Very fast

Advantages:

  • Fast
  • Reliable
  • Energy efficient

Disadvantages:

  • More expensive
4.3 Optical Disk

Backwards compatibility: Blu-ray drives can read DVDs and CDs.

4.4 Magnetic Tape
  • Used for backups
  • Very cheap
  • Slow (sequential access)

5. Storage Media

  • Hard disks → magnetic
  • Optical → laser
  • Flash → electrical charge

6. How Data is Stored (Binary)

All data is stored in binary (0 and 1).

MediaMethod
HDDMagnetic charge
OpticalSurface bumps
FlashElectrical charge
TapeMagnetic polarity
📌 Binary means only two values: 0 and 1.

7. Storage Capacity

  • KB, MB, GB, TB
  • Higher capacity = more data stored

8. Real-World Example

The IBM 350 stored only 3.75 MB, while modern microSD cards can store 256 GB.

💡 Technology has improved: smaller size, bigger storage.

9. Common Exam Mistakes

  • Confusing RAM and storage
  • Forgetting SSD has no moving parts
  • Mixing optical and magnetic storage
