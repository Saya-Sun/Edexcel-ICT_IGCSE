IGCSE ICT Notes

Input Devices

Complete Guide for Exam Success

🔹 What is an Input Device?

An input device is hardware used to send data into a computer system.

  • Allows user interaction
  • Inputs text, images, sound, or data
  • Essential for controlling a computer
📌 Exam Tip: Input devices send data INTO the computer.

1. ⌨️ Keyboard

Used to input text, numbers, and commands.

  • Layouts: QWERTY, AZERTY
  • Supports shortcuts (Ctrl + C, Ctrl + V)
  • Can be wired or wireless
Advantages:
  • Fast for typing
  • Accurate
Disadvantages:
  • Requires practice
  • Not suitable for graphics
💡 Touch typing improves speed and accuracy.

2. 🖱️ Pointing Devices

Used to control the cursor on screen.

Types:
  • Mouse – optical movement detection
  • Trackpad – finger gestures
  • Joystick – gaming control
  • Graphics Tablet – drawing with stylus
📌 Graphics tablet is best for designers.

3. 📄 Scanner (OCR & OMR)

Converts physical documents into digital form.

OCR:
  • Image → Editable text
  • Used in document digitising
OMR:
  • Detects shaded marks
  • Used in MCQ exams
💡 OCR saves time (no retyping needed).

4. 📦 Barcode Scanner

  • Reads product barcodes
  • Includes QR code scanning
Uses:
  • Retail shops
  • Logistics tracking
  • Ticket systems
📌 QR codes store more data than barcodes.

5. 📷 Webcam

  • Captures video/images
  • Used in video calls
  • Can include microphone

6. 🎤 Microphone

Captures sound and converts it into digital data.

  • Uses diaphragm
  • Analogue → Digital conversion
  • Used in voice recognition
📌 Sound must be converted into digital form.

7. 📱 Touch Screen

Types:
  • Resistive – pressure-based
  • Capacitive – multi-touch, more responsive
Comparison:
  • Capacitive = faster
  • Resistive = more durable

8. 🔐 Biometric Scanners

  • Fingerprint
  • Face recognition
  • Iris recognition
  • Voice recognition
Advantages:
  • High security
  • Cannot be forgotten
Disadvantages:
  • Expensive
  • Privacy issues
📌 Iris recognition is the most accurate.

9. 💳 Card Readers

  • Magnetic stripe (least secure)
  • Chip & PIN
  • RFID / NFC (contactless)
💡 Contactless uses NFC technology.

10. 🌡️ Sensors

Automatically collect environmental data.

  • Temperature
  • Light
  • Motion
  • Pressure
Advantages:
  • Continuous monitoring
  • Works in dangerous environments
  • Reduces human error
📌 Used in smart homes and industry.

🧠 Exam Summary

  • Input devices send data INTO the computer
  • Learn examples + uses
  • Compare advantages & disadvantages
  • Know real-life applications
