IGCSE ICT Notes

Input Devices

Complete Guide for Exam Success

🔹 What is an Input Device?

An input device is hardware used to send data into a computer system.

  • Allows user interaction
  • Inputs text, images, sound, or data
  • Essential for controlling a computer
📌 Exam Tip: Input devices send data INTO the computer.

⌨️ Keyboard

Used to input text and commands using keys.

  • Wired or wireless
  • Different layouts (QWERTY, AZERTY)
  • Supports shortcuts (Ctrl + C, Ctrl + V)
Advantages:
  • Fast typing
  • Accurate input
Disadvantages:
  • Needs practice
  • Not suitable for drawing
💡 Touch typing increases speed and accuracy.

🖱️ Pointing Devices

Used to control a pointer on screen.

Types:
  • Mouse: Optical sensor detects movement
  • Trackpad: Detects finger gestures
  • Joystick: Used in gaming
  • Graphics Tablet: Used for drawing with stylus
📌 Exam Tip: Graphics tablet = best for designers

📄 Scanner (OCR & OMR)

Scanners convert paper documents into digital form.

OCR (Optical Character Recognition)
  • Converts images into editable text
  • Used in digitising books
OMR (Optical Mark Recognition)
  • Detects shaded marks
  • Used in MCQ exams
💡 OCR saves time by avoiding retyping.

📦 Barcode Scanner

Reads barcodes to identify products.

  • Linear barcode (lines)
  • QR code (stores more data)
Uses:
  • Shops
  • Parcel tracking
  • Ticket systems
📌 QR codes can be scanned from any angle.

📷 Webcam

  • Captures video and images
  • Often includes microphone
  • Used for video calls and security

🎤 Microphone

Captures sound and converts it into digital data.

  • Uses diaphragm
  • Converts analogue → digital
  • Used in voice recognition
📌 Important: Analogue signal must be converted to digital.

📱 Touch Screen

Types:
  • Resistive: Pressure-based, no multi-touch
  • Capacitive: Uses electrical charge, supports multi-touch
Comparison:
  • Capacitive = more responsive
  • Resistive = more durable

🔐 Biometric Scanners

Use unique human features for identification.

  • Fingerprint
  • Face recognition
  • Voice recognition
  • Iris recognition
Advantages:
  • High security
  • Cannot be forgotten
Disadvantages:
  • Expensive
  • Privacy concerns
📌 Iris scan is the most accurate.

💳 Card Readers

  • Magnetic stripe (least secure)
  • Chip & PIN (secure)
  • RFID / NFC (contactless)
💡 Contactless payment uses NFC technology.

🌡️ Sensors

Automatically collect environmental data.

Examples:
  • Temperature
  • Light
  • Motion
  • Pressure
Advantages:
  • Continuous monitoring
  • Works in dangerous areas
  • Reduces human error
📌 Used in smart homes and industry.

🧠 Exam Summary

  • Input devices send data INTO computer
  • Know examples and uses
  • Compare advantages and disadvantages
  • Understand real-life applications
