🏠 Local Area Networks (LANs)

📌 What is a LAN?

A Local Area Network (LAN) is a network within a small area such as a home, school, or office.

🔗 Network Models

Devices in a LAN can be connected using two models:

  • Peer-to-peer
  • Client-server
1

🤝 Peer-to-Peer Network

Devices share resources directly

Computers share files and resources with each other without using a central server.

💡 Exam Tip: No server used
2

🖥️ Client-Server Network

Uses servers to manage resources

A central server provides services and resources to client computers.

💡 Exam Tip: Server controls the network

✅ Benefits of LANs

1

🖨️ Shared Peripherals

Use devices like printers

Multiple users can access printers and other hardware devices.

2

📂 Shared Storage & Data

Access common files

Users can store and access data from shared locations.

3

🔄 Flexible Access

Use any connected device

Access resources from different computers on the network.

4

🎬 Media Streaming

Stream content easily

Watch videos, listen to music, and play games across devices.

5

💬 Communication

Send messages & files

Users can share files and communicate within the network.

6

🌐 Shared Internet Access

One connection for all

All devices can use a single internet connection.

💡 Exam Tip: LAN = sharing resources + communication
9

🎮 LAN Party

Local multiplayer gaming event

A LAN party is when people connect their computers or consoles on a local area network to play multiplayer games together.

  • Devices are in the same location
  • Connected using a LAN (not internet)
  • Used for multiplayer gaming
  • Fast and low delay (low latency)
💡 Exam Tip: LAN party = local gaming without internet

⭐ Benefits of Client-Server Networks

1

🔐 Access Control

Users can be given or restricted access to resources.

2

🧑‍💼 Centralised Administration

Network is managed by administrators to prevent issues.

3

💾 Centralised Backup

Automatic backups protect user data.

4

⚙️ Shared Software

Applications can be accessed from servers.

5

📂 Shared Storage

Files can be shared with controlled permissions.

6

🌍 Roaming Profiles

Users can log in on any computer and access their files and settings.

💡 Exam Tip: Same files on any computer
