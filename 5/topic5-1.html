🌐 Requirements for Connecting to Networks

📌 Overview

In order to connect to a network, computers must meet certain requirements so they can communicate using standard protocols.

1

💻 Network Operating System (NOS)

Allows computers to communicate on a network

A network operating system provides additional features beyond a normal OS.

  • Passes usernames and passwords to a server for login verification
  • Separates user accounts and protects files
  • Provides access to shared storage and printers
💡 Exam Tip: NOS = security + shared resources

🔍 Device Identification Methods

Devices on a network can be identified using:

  • Internet Protocol (IP)
  • MAC Address
  • Device Name
1

🌐 Internet Protocol (IP)

Unique address used to send data

Each device has an IP address so data can be sent to the correct destination.

📊 Types of IP
  • IPv4: 4 numbers (e.g. 192.168.1.1)
  • IPv6: 8 hexadecimal groups
⚙️ Assignment
  • Manually by network administrator
  • Automatically using DHCP
💡 Exam Tip: IP = identifies destination device
2

🔌 MAC Address

Permanent hardware identifier

A MAC address is assigned to the network interface card (NIC) by the manufacturer.

  • Used in LANs (Wi-Fi, Ethernet, Bluetooth)
  • Usually fixed (cannot change easily)
  • 6 pairs of characters
  • First part = manufacturer
  • Second part = unique device ID
💡 Exam Tip: MAC = physical address (permanent)
3

🖥️ Device Name

Human-friendly name for devices

Device names help users identify computers on a network.

  • Not always unique
  • Not used for communication
  • Can be changed by the user
💡 Exam Tip: Device name = for humans, not for data transfer

📊 Uses of MAC Address

Real-world applications
Use Example
Restricting or allowing access to a network MAC address filtering allows only specific devices to access a network
Identifying a device on a network Wi-Fi hotspots use MAC addresses to track access time limits
Tracking a device Organisations track which access points a device connects to
Assigning static IP addresses DHCP uses MAC address to assign the same IP each time
💡 Exam Tip: MAC address = used for identification, control, and tracking
