🔌 Components of Wired & Wireless Systems

📌 Overview

Wired and wireless systems are made up of different components that allow devices to communicate and share data.

1

🔗 Cable

Connects devices in a wired network
  • Cat5e: 10 / 100 / 1000 Mbit/s
  • Cat6: up to 10 Gbit/s (faster, more expensive)
  • Connects devices via NIC

Fibre Optic: Uses light signals, up to 40 Gbit/s over long distances.

💡 Exam Tip: Network speed = slowest component
2

🔀 Switch

Connects devices and directs data

A switch ensures data reaches the correct device using IP addresses.

💡 Exam Tip: Switch = sends data to correct device
3

🌐 Gateway

Connects different networks

Example: connects a LAN to a WAN.

4

📡 Router

Finds best path for data
  • Stores device addresses
  • Routes data efficiently
  • Often includes switch + Wi-Fi
  • Connects LAN to internet
💡 Exam Tip: Router = connects to internet + routes data
5

📶 Booster

Extends network range
  • Amplifies signals
  • Used for Wi-Fi and wired networks
  • Wireless access points can act as repeaters
6

🖥️ Server

Shares resources with clients

Servers provide shared resources like storage, printers, and applications.

🔐 Authentication Server

Checks usernames and passwords and provides access permissions.

🖨️ Print Server

Manages printers and queues print jobs.

📂 File Server

Provides shared storage for users.

⚙️ Application Server

Runs applications for users over the network.

🌍 Web Server

Handles HTTP requests and delivers web pages.

💡 Exam Tip: Web server = stores websites
