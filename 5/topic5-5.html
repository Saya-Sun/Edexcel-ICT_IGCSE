🔐 Securing Data on Networks

📌 Overview

Security protects network data and resources from unauthorised access.

1

🔑 Logins & Passwords

Authenticates users
  • Users must log in to access network resources
  • Password verifies identity
  • Authentication servers manage access
💡 Exam Tip: Authentication = checking identity
2

🔥 Firewall

Controls network traffic
  • Blocks unauthorised access
  • Monitors incoming and outgoing data
  • Can be hardware or software
3

🔒 Encryption

Scrambles data for security

Encryption makes data unreadable to unauthorised users.

I

🔁 Symmetric Key Encryption

Same key used to encrypt and decrypt
⚙️ How it works
  1. Sender encrypts data using a secret key
  2. Encrypted data is sent
  3. Receiver uses the same key to decrypt
✅ Advantages
  • Fast and efficient
  • Uses less processing power
⚠️ Disadvantages
  • Key must be shared securely
  • If key is stolen → data compromised
💡 Exam Tip: Symmetric = 1 key (fast but less secure)
II

🔑 Public Key Encryption

Uses two keys (public + private)
⚙️ How it works
  1. Receiver shares public key
  2. Sender encrypts data using public key
  3. Data is sent securely
  4. Receiver decrypts using private key
✅ Advantages
  • No need to share private key
  • More secure for communication
⚠️ Disadvantages
  • Slower than symmetric encryption
  • Uses more processing power
💡 Exam Tip: Public key = encrypt, Private key = decrypt

⚖️ Quick Comparison

Feature Symmetric Public Key
Number of keys One key Two keys
Speed Fast Slower
Security Less secure More secure
Key sharing Must be shared Public key shared safely
💡 Exam Tip: Symmetric = fast, Public key = secure
III

📶 WEP

Basic wireless security

Uses the same key for all devices → less secure.

IV

📡 WPA

Improved wireless security

Uses different keys for devices and data packets.

4

🌐 VPN (Virtual Private Network)

Secure connection over the internet

A VPN allows users to access a private remote network securely over the internet.

  • Encrypts data
  • Hides the user's real IP address
  • Shows the VPN server’s IP instead
  • Used for privacy and remote access
💡 Exam Tip: VPN = encryption + hides real IP
5

📂 File Access Rights

Controls who can access and modify files

File access rights (permissions) determine what a user can do with files, folders, or drives.

  • Read: view files only
  • Write: edit or add data
  • Delete: remove files
  • Admin: full control (change permissions, manage users)

Permissions can be set for individual users or groups to protect data and prevent unauthorised changes.

💡 Exam Tip: Access rights = control + data security
6

📜 Transaction Logs

Records network activity

Helps detect unusual or suspicious activity.

7

💾 Backups

Protects data from loss

Copies of files stored separately for recovery.

💡 Exam Tip: Backup = data recovery
