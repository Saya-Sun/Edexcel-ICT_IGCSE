🌐 Connecting to and Using the Internet

📌 Overview

To access online services, users need an internet connection and software to use services safely and effectively.

1

🌍 Internet Service Provider (ISP)

Provides access to the internet

Users must subscribe to an ISP to connect to the internet.

  • Uses mobile and landline networks
  • Usually requires a subscription fee
  • Some provide free access (community schemes)
💡 Exam Tip: ISP = connects user to internet infrastructure
2

🌐 Web Browser

Software to access web pages

A web browser requests and displays data from web servers.

  • Examples: Chrome, Firefox, Edge, Safari, Opera
  • Available on computers and mobile devices
💡 Exam Tip: Browser = used to view web pages
3

🔍 Search Engine

Finds information using keywords

Users enter keywords and the search engine returns matching results.

💡 Exam Tip: Search engine ≠ browser (browser accesses, search engine finds)
4

🚫 Filter Software

Blocks inappropriate content

Filter software compares websites with stored lists.

  • Blacklist: blocked websites
  • Whitelist: allowed websites
  • Match blacklist → blocked
  • Match whitelist → allowed
  • No match → allowed
💡 Exam Tip: Used in schools to protect users
