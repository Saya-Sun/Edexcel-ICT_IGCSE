📘 Memory & Processors

⚡ RAM
Temporary, volatile memory used for running programs.
  • Read & write
  • Fast access
  • Stores active programs
  • Lost when power off
📊 RAM Size Impact
  • More RAM → better multitasking
  • Less RAM → uses virtual memory (slow)
  • Virtual memory uses secondary storage
📀 ROM
Permanent, non-volatile memory storing startup instructions.
  • Stores BIOS / boot program
  • Used in embedded systems
  • Data not lost when power off
📊 RAM vs ROM Comparison
RAM ROM (all types)
Stores data when power off? No (volatile) Yes (non-volatile)
Can swap instructions in and out? Yes No
Accessible in any order? Yes Yes
Intended to store... Temporary data Permanent data
Can be upgraded? Yes No
👉 RAM = temporary & changeable | ROM = permanent & fixed

Types of ROM:

  • PROM → written once
  • EPROM → erased using UV light
  • EEPROM → erased electrically
👉 EEPROM can be updated without removing the chip
🔌 EEPROM & Flash
  • EEPROM can be erased and rewritten
  • No need to remove chip
  • Used for firmware updates
  • Flash memory developed from EEPROM
💽 Flash Memory
Non-volatile memory with no moving parts.
  • Fast access
  • Low power consumption
  • Used in USB, SSD, SD cards
🧮 CPU
  • Executes instructions
  • Controls operations
  • Multi-core improves performance
⏱️ Processor Speed
  • Measured in Hz (GHz, MHz)
  • Performance depends on cores + work per cycle
